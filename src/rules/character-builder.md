# Character Builder

<!--{ "parent": "core", "order": 3.5 }-->

The interactive, rule-enforcing character builder — the same rules as [Creating Characters](/creating-characters/), live. Set attributes, add background factors in life-story order, buy skills and fields; point totals, slot budgets, and level caps are enforced as you go. Characters save to this browser (localStorage) and export to JSON. Toggle **setting tags** to hide content you're not using (e.g. keep SF gear off a fantasy sheet).
