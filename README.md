# ForeSight 2026
<!--{ "pin": "top", "headTitle": "ForeSight 2026 — a streamlined tabletop RPG", "description": "A modern edition of the ForeSight RPG: one resolution rule (Score × Ease Factor), lethal fast combat, and a careful generative magic system." }-->

A streamlined, modern edition of the **ForeSight** tabletop RPG, built on the unpublished 2004 second edition and the 1986 original.

**One rule:** Success Chance = Score × Ease Factor — roll D100 under it. Everything else is that rule applied to different situations.
