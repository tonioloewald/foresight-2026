# ForeSight 2026

A streamlined, modern edition of **ForeSight**, the general-purpose tabletop RPG first published in 1986. This edition builds on the unpublished 2004 second-edition draft and the original, distilling four decades of play into a set of one-page rules around a single resolution mechanic.

> **One rule:** Success Chance = Score × Ease Factor. Roll D100 under it. Everything else is that rule applied to different situations.

## Design philosophy

Keep the mechanics simple, push the probabilities in the right general direction with the right general magnitude, and let the results speak for themselves. ForeSight doesn't simulate reality precisely; it makes believable things likely and absurd things rare, using rules you can run from memory — and it stays *transparent*, so the setting seen through the rules is just the setting.

## Status

Work in progress, in active playtesting. Rules text, costs, and balance numbers are provisional and subject to change at the table.
